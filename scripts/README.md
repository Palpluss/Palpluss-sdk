# Scripts

Tooling and automation scripts for the monorepo.

## Planned scripts

| Script | Purpose |
|---|---|
| `validate-openapi.sh` | Validate `openapi/palpluss-v1.yaml` against the OpenAPI 3.1.0 schema |
| `check-contract.ts` | Verify TypeScript types match the OpenAPI contract |
| `sync-versions.sh` | Utility to inspect version state across all packages |

Scripts will be added here as the monorepo matures.
